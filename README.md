# SMS Spam Classification using NLP

## Overview
This project focuses on detecting spam SMS messages using Natural Language Processing (NLP) and machine learning techniques. The goal of the project is to classify messages into two categories: Spam and Ham (not spam).

The text messages are converted into numerical features using NLP techniques and then used to train a machine learning model for classification.


## Dataset
The dataset contains SMS messages labeled as either spam or ham.
The dataset contains around 5572 SMS messages with two columns:

Each record in the dataset contains:
- Category (Spam or Ham)
- Message text

The dataset used in this project is stored in the repository as:
sms_spam_dataset.csv


## Technologies Used
The project was implemented using the following tools and libraries:

- Python
- Pandas
- NumPy
- Scikit-learn
- Natural Language Processing (NLP)


## Project Workflow
The main steps followed in this project are:

1. Load the dataset
2. Perform basic data cleaning
3. Text preprocessing
4. Convert text into numerical features using vectorization
5. Train the machine learning model
6. Evaluate the model performance


## Files in this Repository
sms_spam_classification_nlp.ipynb – Jupyter notebook containing the full implementation  
sms_spam_dataset.csv – Dataset used for training the model


## Conclusion
This project demonstrates how NLP techniques can be used to automatically detect spam messages from SMS text data using machine learning.
